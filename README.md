# SmartSpend

SmartSpend is an AI-powered financial awareness web application designed for students and first-time earners.

## Features
- Automatic expense categorization
- Pie chart spending visualization
- AI-generated budget insights
- Data persistence using browser storage

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Chart.js

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Open:
   http://127.0.0.1:5000
